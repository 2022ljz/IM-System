# 或许是你的第一个基础 Go 项目：Server / Client 通信系统（TCP Chat）

这是一个用 Go 语言实现的**基础服务器-客户端通信项目**，通过 TCP 建立连接，实现一个简易的命令行聊天系统。  
适合作为 Go 网络编程入门练手项目：从“能连上、能发消息”，到“多人在线、私聊、公聊、改名、超时下线”，把常见功能走一遍。

---

## 项目亮点

- 命令行聊天室：开箱即用，启动后即可聊天
- 支持多人在线：进入即上线、退出即下线
- 支持公聊 / 私聊：群发和点对点消息都可用
- 支持查看在线用户：快速列出当前在线名单
- 支持修改用户名：随时改名并同步到在线列表
- 空闲超时自动踢出：长时间不操作会被服务端断开连接（更贴近真实服务）

---

## 功能说明

### 客户端（Client）
客户端启动后提供菜单模式：

- `1` 公聊模式：输入内容直接发送到聊天室
- `2` 私聊模式：选择在线用户后发送私聊消息
- `3` 修改用户名：更新当前用户名
- `0` 退出

常用指令（由客户端自动发送到服务端）：
- `who`：查看在线用户列表
- `rename|新名字`：修改用户名
- `to|对方用户名|消息内容`：私聊

### 服务端（Server）
服务端负责：

- 接收客户端连接
- 维护在线用户
- 将消息转发给其他用户
- 处理基础命令（who / rename / to）
- 空闲超时断开连接（默认 120 秒）

---

## 运行环境

- Go 1.18+
- 本地运行（默认监听 `127.0.0.1:8080`）

---

## 快速开始

### 1）启动服务端

进入 server 目录：

```go build run.exe main.go server.go user.go ```
```.\run.exe ```

### 2）启动客户端（建议多个）

进入 client 目录：

```go run client.go```
